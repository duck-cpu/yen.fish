Dog Breeder Landing Page


    This landing page is built with React.js, incorporating Framer Motion to create smooth, engaging animations. The site showcases available puppies using a responsive Flexbox layout, ensuring a seamless and visually appealing browsing experience. A dedicated review section highlights customer feedback, adding credibility and fostering trust with potential buyers.

    The site is designed to be easily extendable, with integration capabilities for a flat file database or an API. This enables straightforward management of puppy inventory and customer reviews, allowing users to add or remove puppies and update or upload new reviews with minimal effort.

    There is future plans to implement this as well as a contact page, and make the site responsive.
